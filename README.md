# Prajwal Shetty
## Full Stack Developer | Node.js | AI Explorer

Personal portfolio showcasing full-stack web products, machine learning projects, and practical AI experiences.

Built with vanilla HTML, CSS, and JavaScript. Responsive, accessible, and themeable (dark/light).

---

## Quick Start

1. Open `index.html` in your browser.
2. Navigate sections: Home, About, Projects, Journey, Contact.
3. Toggle theme via header button.
4. Explore project pages in `/projects`.

```bash
# Windows
start index.html

# Or drag index.html to browser
```

---

## Features

- Responsive design (mobile-first, grid/flex layouts)
- Dark/light theme toggle with localStorage
- Smooth scroll animations (IntersectionObserver)
- Mobile navigation (hamburger, ARIA accessible)
- Glassmorphism UI with backdrop filters and gradients
- Keyboard navigation and skip link
